# Kitchen Inventory & Ordering System

Цей проєкт реалізує консольну систему управління товарами кухонного асортименту та замовленнями користувачів. Він дозволяє додавати, редагувати та відслідковувати товари, створювати та оплачувати замовлення, а також керувати користувачами.

---

## Особливості

- **Керування товарами**:
  - Додавання, редагування та вивід товарів.
  - Категорії: `Cookware`, `Tableware`, `Cutlery`, `Containers`, `Appliances`, `CleaningSupplies`.
  - Валідація даних (ціна > 0, кількість ≥ 0, унікальне ім’я).

- **Користувачі**:
  - Реєстрація користувачів із валідацією імені, прізвища, email, пароля та ролі.
  - Ролі користувачів: `Admin`, `Customer`.

- **Замовлення**:
  - Додавання продуктів до замовлення з автоматичним обчисленням загальної суми.
  - Валідація кількості та продукту.
  - Статуси замовлень: `Created`, `Paid`, `Canceled`.
  - Методи `PayOrder` та `CancelOrder` для управління замовленнями.
  - Оновлення часу останньої зміни (`UpdatedAt`) при зміні замовлення.

- **Безпека**:
  - Пароль користувача з мінімальною довжиною 6 символів.
  - Email перевіряється на коректний формат.
  - Всі enum-поля мають валідацію на допустимі значення.

- **Вивід у консоль**:
  - Метод `ForTable()` для красиво форматованого відображення користувачів, товарів та замовлень у таблиці.
  - Вивід детальної інформації про товари в замовленні.

---

## Структура проекту

- `Product` – клас товару.
- `ProductCategory` – перелік категорій товарів.
- `User` – клас користувача з ролями (`UserRole`).
- `Order` – клас замовлення.
- `OrderProduct` – клас для зв’язку товарів із замовленням.
- `Enums`:
  - `ProductCategory`
  - `UserRole`
  - `OrderStatus`

---

## Приклади використання

```csharp
// Створення користувача
var user = new User("Іван", "Іваненко", "ivan@example.com", "password123", UserRole.Customer);

// Створення продуктів
var pan = new Product("Frying Pan", "Non-stick frying pan", 10, 25, ProductCategory.Cookware);
var knife = new Product("Knife Set", "6-piece knife set", 5, 55, ProductCategory.Cutlery);

// Створення замовлення
var order = new Order(user);
order.AddProduct(pan, 2);
order.AddProduct(knife, 1);

// Оплата замовлення
user.PayOrder(order.Id);

// Вивід інформації про замовлення
Console.WriteLine(order.ForTable());
Console.WriteLine(order.ItemsForTable());
